# Tic Tac Toe Duel

This is a static browser game package for CaSkill creator uploads.

## Package contents

- `manifest.json`
- `README.md`
- `dist/index.html`
- `dist/styles.css`
- `dist/script.js`

## Gameplay

- Two local players share the same screen.
- Player X starts first.
- Players click empty cells to place marks.
- First player to align three marks wins the round.
- Use the reset button to start a new round.
